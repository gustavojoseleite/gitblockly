Blockly Git

Esse é um simples gerador interativo de códigos para Git. Com ele é possível praticar os principais comandos do Git através da ferramenta Blockly. O aprendizado é baseado no agrupamento de blocos de modo a ter uma sintaxe válida para comandos utilizados no Git. Além disso, o usuário pode visualizar maiores informações a respeito do comando ou opção posicionando o cursor sobre o bloco. Por fim, ao término da junção dos blocos, basta clicar sobre o botão Gerar e um código correspondente aos blocos é gerado para que possa ser utilizado no próprio Git.
